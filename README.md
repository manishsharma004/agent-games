# Agent Games

Browser game playground built with React, TypeScript, Vite, and Tailwind.

## Package manager

Use **Bun** for all local development and automation tasks in this repository.

## Scripts

```bash
bun install
bun run dev
bun run build
bun run lint
bun run preview
```

## Notes for contributors and agents

- Do not use `npm`, `yarn`, or `pnpm` in this repository.
- Use `bun install` to sync dependencies and lockfile.
- Use `bun run <script>` for all scripts.
